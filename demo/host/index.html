DEMO This page lives on a distinct origin from the widget service (port 8090 vs 8080) to prove the allowlist is real.

The Maintainer's Copilot widget appears in the bottom-right corner. It loads only because http://localhost:8090 is in this widget's allowed_origins list. Change the origin in the widget admin and the embed will be blocked by the Content-Security-Policy: frame-ancestors header the api sets.

How the allowlist works

  1. Admin creates a widget config via the Streamlit admin page, setting allowed_origins: ["http://localhost:8090"].
  2. Host page pastes the <script> snippet.
  3. Loader injects an <iframe> pointing at the widget service.
  4. Widget fetches /embed/config/{id} from the api; the response carries Content-Security-Policy: frame-ancestors http://localhost:8090.
  5. If this page's origin is not in the list, the browser blocks the iframe.
